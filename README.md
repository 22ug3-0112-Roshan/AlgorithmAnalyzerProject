## ğŸš€ **Project Overview**
This project evaluates **execution time** of different algorithms using Java.  
Each team member implemented one algorithm, executed it on input sizes:

- **100**
- **500**
- **1000**

Time was measured using `System.nanoTime()` for accuracy.

The aim is to understand:
- Time complexity  
- Practical runtime behavior  
- Differences between searching & sorting algorithms  
- How input size impacts performance
